Vercel

För att publicera uppgiften har jag valt att använda mig av Vercel. Vercel är en molnplattform för statiska sajter som är väldigt lätt att använda, går att använda gratis och kan kopplas direkt med ett GitHub-konto.
För att publicera denna uppgift så kopplade jag mitt vercel-konto till det Git-repository som uppgiften låg på. Med koppling till rätt mapp krävdes det sedan bara ett klick för att allt skulle läggas över på en egen publicerad hemsida. Jag kommer verkligen att använda mig av Vercel igen, då detta var så mycket lättare än mitt vanliga sätt med FileZilla.

Vill man använda Vercel när man jobbar i team måste man använda pro-versionen. Denna version kostar ca 200 kr i månaden. Vercel finns även som paket till hela företag.

Vercel har ett brett kod-stöd, och stödjer programmeringsspråk såsom: HTML, CSS, JavaScript, Ruby, Python och TypeScript.
